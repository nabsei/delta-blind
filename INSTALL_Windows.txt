DELTA BLIND — BETA (Windows)
==============================

This is an early beta build, not yet code-signed. Windows SmartScreen may
show a warning the first time you run it — this is normal for an early
indie plugin build, here's how to get past it.

INSTALL
-------

1. Copy the "Delta Blind.vst3" folder to:
   C:\Program Files\Common Files\VST3\

2. Rescan plugins in your DAW (or restart it) — Delta Blind should now
   show up under your effects.

   (You can also run "Delta Blind.exe" directly — no DAW needed, use
   its built-in TEST SIG button to try it without routing real audio.)

USING IT
---------

Delta Blind needs TWO audio sources to compare: the main input (source A)
and a sidechain input (source B) — most DAWs let you route a sidechain
from another track onto an insert. Pick A or B to hear it, toggle MATCH
on to have B's level automatically trimmed to meet A's, so a preference
between them reflects a real difference, not just "louder sounds better".

No sidechain routed yet? Click the TEST SIG button in the plugin UI — it
feeds a built-in A/B pair (B is +7dB louder than A) so you can hear MATCH
working immediately, no setup required.

IF WINDOWS SMARTSCREEN BLOCKS IT
---------------------------------

When you first run the .exe (or your DAW loads the plugin), you may see
"Windows protected your PC". Click "More info", then "Run anyway".
This appears because the build isn't yet signed with a paid code-signing
certificate — it does not mean the file is unsafe, just that Windows
doesn't yet recognize the publisher.

QUESTIONS / BUGS
-----------------

This is a beta. If something sounds wrong or the matching/UI misbehaves,
please report it — beta feedback directly shapes what ships in v1.
